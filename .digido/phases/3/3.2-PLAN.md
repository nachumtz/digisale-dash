# Phase 3.2: עיצוב Dark Corporate + UX Polish

**מטרה:** לתת לדשבורד מראה עסקי מקצועי ומלוטש — מעבר ל-Default Streamlit.

**קבצים מעורבים:**
- `app.py` — הוספת Custom CSS
- `.streamlit/config.toml` — עדכון אם נדרש

**Tasks:**
- [ ] הוסף ב-`app.py` Custom CSS (באמצעות `st.markdown`) לסגנון: Header עם border תחתית, כרטיסי KPI עם `border-radius` ו-`box-shadow`, רקע כהה לאזורי תוכן
- [ ] הוסף כותרת ראשית עם לוגו/אייקון (`📊 Digisale-Dash | דשבורד מכירות`) ושם החברה כ-Subtitle
- [ ] ודא UX: Sidebar מסודר עם כותרת "📁 העלאת קבצים", הוראות Upload ברורות בעברית, Spinner בזמן עיבוד

**הערות:**
- אין צורך בספריות CSS חיצוניות — רק `st.markdown(..., unsafe_allow_html=True)`
- שמור על עקביות: כל הטקסט הפנים-ממשקי בעברית

**Success Criteria:**
- הדשבורד נראה Corporate/Professional לעומת Default Streamlit
- כרטיסי KPI בולטים ויזואלית בהשוואה לגרפים
- Sidebar מסודר ומדריך את המשתמש בבירור
