# Phase 1.2: מודול עיבוד נתונים

**מטרה:** לבנות את מנוע ה-Data (טעינה, Merge, Validation, חישוב KPIs) כמודול נפרד ומנותק מה-UI.

**קבצים מעורבים:**
- `data_loader.py` — חדש (מודול מרכזי לכל עיבוד הנתונים)

**Tasks:**
- [ ] כתוב פונקציה `load_and_validate(file, expected_columns)` שטוענת CSV/Excel ומאמתת עמודות
- [ ] כתוב פונקציה `merge_data(orders_df, customers_df, products_df)` שמבצעת Left Join כפול (על Customer_ID ועל Product_ID) ומחשבת `Revenue` ו-`Profit` לכל שורה
- [ ] כתוב פונקציה `calculate_kpis(merged_df)` שמחזירה dict עם: `total_revenue`, `total_profit`, `completed_orders`, `cancellation_rate`

**הערות:**
- `Revenue = Unit_Price × Quantity × (1 - Discount)`
- `Profit = (Unit_Price - Cost_Price) × Quantity × (1 - Discount)`
- KPIs (חוץ מ-cancellation_rate) = על `Status == "הושלם"` בלבד
- `cancellation_rate = COUNT(בוטל) / COUNT(total) × 100`

**Success Criteria:**
- טעינת קבצי `*_deme.csv` מחזירה DataFrames ללא שגיאה
- `merge_data` מחזיר DataFrame עם עמודות `Revenue` ו-`Profit`
- `calculate_kpis` מחזיר dict עם 4 מפתחות ומספרים הגיוניים
