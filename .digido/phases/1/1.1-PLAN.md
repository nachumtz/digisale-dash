# Phase 1.1: הקמת הפרויקט

**מטרה:** יצירת סביבת Python מבודדת, מבנה תיקיות, והגדרות Streamlit — כך שהפרויקט יהיה מוכן לפיתוח.

**קבצים מעורבים:**
- `requirements.txt` — חדש
- `.streamlit/config.toml` — חדש (Dark theme)
- `app.py` — שלד ריק

**Tasks:**
- [ ] צור `venv` בתיקיית הפרויקט והתקן packages: `streamlit`, `pandas`, `plotly`, `openpyxl`
- [ ] צור `requirements.txt` עם כל הגרסאות (pip freeze)
- [ ] צור `.streamlit/config.toml` עם הגדרות Dark theme ו-`layout=wide`
- [ ] צור `app.py` שלד עם `st.set_page_config` ו-`st.title` בסיסי

**הערות:** יש להריץ את הכל בתוך ה-venv. עבור Windows, הפעלת venv: `venv\Scripts\activate`.

**Success Criteria:**
- `streamlit run app.py` מריץ בדפדפן ומציג מסך כותרת ריק
- Dark theme פעיל
- כל ה-imports עובדים ללא שגיאות
