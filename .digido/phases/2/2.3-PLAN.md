# Phase 2.3: גרפי Plotly

**מטרה:** להוסיף 2 גרפים אינטראקטיביים של Plotly מתחת ל-KPIs, עם Hover מדויק.

**קבצים מעורבים:**
- `charts.py` — חדש (מודול גרפים)
- `app.py` — import וקריאה ל-charts.py

**Tasks:**
- [ ] ב-`charts.py` כתוב `chart_revenue_by_category(df)` — Bar Chart של `Revenue` לפי `Category` (גרסה מסוננת: `Status==הושלם`)
- [ ] ב-`charts.py` כתוב `chart_revenue_by_segment(df)` — Bar Chart של `Revenue` לפי `Customer_Segment` (סינון זהה)
- [ ] ב-`app.py` הצג 2 עמודות (`st.columns(2)`) עם הגרפים זה לצד זה, עם כותרות ברורות בעברית

**הערות:**
- השתמש ב-`plotly.express.bar` עם `template="plotly_dark"` (מתאים ל-Dark mode)
- הוסף `hover_data` שמציג את הסכום המדויק ב-₪ בהעברת עכבר
- ציר Y: ערכים בפורמט קריא (K/M אוטומטי של Plotly)

**Success Criteria:**
- 2 גרפים מוצגים זה לצד זה
- Hover מציג את שם הקטגוריה/סגמנט + הסכום ב-₪
- Dark theme תואם לממשק הכללי
- הגרפים מגיבים לנתונים האמיתיים (לא hardcoded)
